# NetCDF Examples

A series of toy examples for NetCDF demonstrations accompanying the talk:
[A Random Walk through NetCDF](https://jackatkinson.net/slides/netcdf-2026/) by
[Jack Atkinson](https://jackatkinson.net).

The repository consists of a series of CDL (NetCDF Common Data Language) files that
can be used to generate NetCDF files.

We assume that [NetCDF](https://github.com/Unidata/netcdf-c) is installed on the system
and the `ncgen` utility is available.

NetCDF files can be generated by running the enclosed shell script:
```bash
./generate.sh
```
